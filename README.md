findsorguk
==========

New repository for next generation of the PAS website

Database
========

This uses MySQL for database storage and is version 5+ compatible

Requirements
============
Operating system requirements:

* PHP 5.2+
* Solr 4.2
* MySQL 5+
* APC Cache
* Curl
* ImageMagick
* GD
* PDO

PHP libraries

* Zend Framework 1.12+
* Solarium 2
* ZendX_JQuery
* HTMLPurifier
* EasyBib
* Tcpdf
